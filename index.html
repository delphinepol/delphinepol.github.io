English version

J'ai obtenu mon doctorat de mathématiques à l'Université d'Angers en décembre 2016. J'ai ensuite été postdoctorante à l'Université de Hokkaido au Japon, puis à l'Université Technique de Kaiserslautern en Allemagne.

Domaines de recherche

Mes travaux de recherche réalisés pendant ma thèse et mes postdocs portent sur :
  • un type particulier de singularités appelées "singularités libres"
  • un cas particulier de singulartités libres : les courbes
  • les arrangements de sous-espaces vectoriels
  • des singularités provenant de graphes

Programmation

D'octobre 2020 à septembre 2021, j'ai été développeuse pour le logiciel de calcul formel OSCAR, écrit en Julia. Mon rôle était de :
  • développer une structure pour les courbes algébriques planes
  • implémenter des fonctions pour travailler avec les courbes
  • écrire la documentation correspondante
  • écrire des tests afin de permettre l'intégration continue via GitHub Actions
Dans le cadre de mes travaux de recherche, j'ai utilisé le logiciel Singular. J'ai développé la bibliothèque suivante pour Singular, en collaboration avec Raul Epure :
  • logmodules.lib
